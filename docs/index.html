1.1 Tables

1.1.1 Adjust column width with knitr::kable()

html_table_width() function:

html_table_width <- function(kable_output, width){
  width_html <- paste0(paste0('<col width="', width, '">'), collapse = "\n")
  sub("<table>", paste0("<table>\n", width_html), kable_output)
}

https://github.com/rstudio/bookdown/issues/122

1.1.2 Line breaks in table cells with knitr::kable()

best option

add <br>

x <- data.frame(a = "asdf asdf", b = "asdf<br>asdf")
knitr::kable(x)
a b
asdf asdf asdf
asdf

1.1.3 Line breaks in table cells with knitr::kable(), using format="html"

Add escape=FALSE

x <- data.frame(a = "asdf asdf", b = "asdf<br>asdf")
knitr::kable(x, format="html", escape=FALSE)
a b
asdf asdf asdf
asdf
